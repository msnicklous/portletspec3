This document describes the JSR 362 Portlet Specification 3.0 Technology Compatibility Kit (TCK). The TCK helps portal vendors test compatibility with the Portlet Specification by providing test cases that prove compliance with the specification.

The TCK implements the assertions contained in the JSR 362 TCK Assertions document.

The README.txt file contained in the project base directory explains how the TCK can be used.

This is a work in progress and is subject to change at any time.
